== Description ==

<p>Require payments on your Contact Form 7 forms with PayPal or Stripe payment gateways.</p>

<p>The lite version allows you to configure payments for one of your contact forms, created with Contact Form 7, and charge custom amounts and currencies with PayPal and Stripe payment gateways.</p>

<p>Loving the plugin so far? You may upgrade to the full version of <a href="https://www.wputil.com/cf7-payments">Contact Form 7 PayPal & Stripe Payments</a>, for a lifetime license!</p>

== Installation ==

* Install and activate the plugin:

1. Upload the plugin to your plugins directory, or use the WordPress installer.
2. Activate the plugin through the \'Plugins\' menu in WordPress.
3. Once done, use the plugin settings link to access settings, or go to Dashboard > Settings > CF7 Payments
